Primer paso:
Conectamos nuestro repositorio local del eclipse con el repositorio
 remoto del GitHub (clonando el contenido de este).

Segundo paso:
Ponemos nuestros datos en el fichero CONTRIBUTING.md

Tercer paso:
Cambiamos la URL de tu repositorio remoto por el de nuestro repositorio remoto

Cuarto paso:
Generamos un fichero README.md

Quinto paso:
Subimos los cambios al repositorio remoto haciendo el add y el commit
desde el Eclipse
